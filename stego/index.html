Things to check before starting

  • CSS can highlight 'nth' characters in a string
    • Possible if you loop through binary array and output everything in spans with their own classes
  • JS can convert binary into integer
    • Can use parseInt() with a radix argument
  • JS can convert binary into ASCII
    • Make string that you're hiding a string of ASCII char codes, encoded in binary, then convert back using String.fromCharCode()
